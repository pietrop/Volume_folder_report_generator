Volume report generator

Use this os x app to generate a report of project files and audio files that might be missing from the video servers.

Usage

Launch the app Select Volume Folder to set the folder path of the folder that contains the video projects folders. Saving the report to desktop generates a report on your desktop. Video server folder structure

The app assumes there is the following folder structure on the video server.

└── Volume
       ├── SAMPLE_Project_1_folder
       │   └── ...
       └── SAMPLE_Project_1_folder
             └── ...
